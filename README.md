# Requisitos para rodar o projeto:
-Nodejs ;<br/>
-Npm;<br/>
-Back end rodando com o docker.<br/>

## Passo a passo:

Clone Repositório
```sh
git clone https://github.com/HugoVersiani/front-teste-tec.git front-teste-tec
```
```sh
cd front-teste-tec

```

Instale o projeto e as dependencias:
```sh
npm install
```

Rode o projeto

```sh
npm run dev
```
